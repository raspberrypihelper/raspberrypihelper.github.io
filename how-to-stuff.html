Raspberry Pi zero Fm transmitter (Only works with raspberry pi zero)


Download the .img file here, use Berryboot to install it onto your raspberry pi zero. Then download Cyberduck (mac) to your computer. Boot your raspberry pi up to the .img file you downloaded and login if needed as: pi (username) raspberry (password). Then go to your computer and open the Cyberduck (mac) or Putty (Windows) and ssh into your raspberry pi zero. Then transfer your .mp3 music files to the directory: /home/pi/Music/
Then get onto your raspberry pi zero and go into the terminal. Type this in, but edit the text to play your own files:
sudo /usr/bin/mpg123 -4 -s -Z /home/pi/Music/Test.mp3 | sudo /home/pi/pifm/pifm - 88.3
I have included a test.mp3 file, so that you can test the transmitter. Then solder a antenna onto PIN 4 of your raspberry pi zero. The default broadcasting fm frequency is 88.3, as you can see in the command. You need to replace the "Test.mp3" with your song name. The command is case sensitive, so make sure you type the song name exactly. 
P.S. You can easily list the songs that you transferred to your pi by using these commands: cd /home/pi/Music/ 
then these two letters: ls
You can also change the audio output of the raspberry pi zero. Say, you have a usb audio dongle, or a module that connects to the GPIO header. You can change the audio output by the following command: sudo raspi-config
Navigate to the "advanced options" by using your arrow keys, then click "enter". Scroll to the "Audio" option and select which audio output source you want to use. ​

Getting Nextcloud (Personal Storage Service) onto your Raspberry Pi using Berryboot

1. Download this image.
2. Copy it to a usb flash drive.
3. Boot up your Raspberry Pi on berryboot.
4. Click menu.
5. Click and hold mouse over "Add OS".
6. Click add from USB Drive.
7. Select usb drive.
8. Navigate to the .img file's location.
9. Double click on .img and it will copy to your Raspberry Pi.
10. Once done copying, click exit.
11. Boot into NextCloud OS. 
12. Make sure you are connected to wifi or ethernet.
13. Find out your Raspberry Pi's ip address.
14. Type in your Raspberry Pi's ip address in a computer. It should look something like this: 192.168.1.253
15. Add a / after the ip address. Also type the word "Nextcloud" after the "/".
16. Create user account, then download Nextcloud sync app from here.

PoisonTap for Raspberry Pi Zero (Only works with the Raspberry Pi Zero!)

You Can read more about PoisonTap here.
You can get PoisonTap by doing the following: 
1. Download the image from here.
2. Copy it to a usb flash drive.
3. Boot up your Raspberry Pi on berryboot.
4. Click menu.
5. Click and hold mouse over "Add OS".
6. Click add from USB Drive.
7. Select usb drive.
8. Navigate to the .img file's location.
9. Double click on .img and it will copy to your Raspberry Pi.
10. Once done copying, click the "Set as Default" button to allow the Pi to boot up automatically to PoisonTap OS after 10 seconds.
11. Click exit.
12. Plug the raspberry Pi into a locked PC or Mac. You will need a standard micro usb cable. You need to plug the micro usb side of the cable into the USB otg (Data) plug on the Pi. (Where you usually plug in your otg cable and usb hub)
13. Hack Away!!

Installation of RTL892EU wifi dongle driver

1. Go to this link on your Raspberry Pi.
2. Download the "install.sh" file.
3. Go to your favorite terminal and do: "sudo sh /path/to/sh/file.sh"
4. Example:  sudo sh ~/detectingdragon77/Downloads/install.sh
5. It will automatically compile and install the driver.
